# Signal JS

Signal JS — статический выпускной проект по дисциплине «Веб-дизайн».  
Это образовательная платформа для диагностики и тренировки знаний по JavaScript.

## Что реализовано

- 4 теста по одной предметной области — JavaScript
- в каждом тесте по 20 вопросов
- 2 режима прохождения:
  - **Тренировка** — после ответа сразу показывается разбор
  - **Экзамен** — разбор доступен после завершения теста
- стартовый экран с анимацией логотипа и первичной настройкой локального профиля
- таймер
- прогресс прохождения
- локальный профиль без бэкенда
- история попыток
- экспорт одной попытки в CSV
- экспорт всей истории в CSV
- сохранение данных через `localStorage`
- адаптивный светлый интерфейс

## Стек

## Как запустить локально

1. Открой папку проекта в VSCodium.
2. Запусти локальный сервер.
   - например, через расширение Live Server
   - или через команду `python -m http.server`
3. Открой `index.html` в браузере через локальный сервер.

> Желательно не открывать файл двойным кликом как `file://`, а использовать локальный сервер, потому что данные тестов загружаются из `data/tests.json`.

## Как выложить на GitHub Pages

1. Создай новый репозиторий на GitHub.
2. Загрузи все файлы проекта в репозиторий.
3. Перейди в **Settings → Pages**.
4. В пункте **Build and deployment** выбери публикацию из ветки.
5. Укажи нужную ветку и корневую папку.
6. Сохрани настройки и дождись публикации.
